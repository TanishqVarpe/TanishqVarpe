### 📂 Featured Projects

#### 🩺 [Urethritis Diagnosis Tool](https://github.com/TanishqVarpe/Urethritis-CEP-Project.git)
A full-stack clinical decision-support tool that analyzes patient data to suggest likely diagnosis outcomes.
- **Stack:** Java Spring Boot (backend), React.js (frontend), FastAPI (ML inference service)
- Designed a multi-service architecture connecting a Spring Boot REST API with a Python-based ML microservice
- Built to help demonstrate how ML models can be integrated into real clinical workflows

#### ✋ [Hand Gesture Recognition with TensorFlow](https://github.com/TanishqVarpe/Hand-Gesture-using-Tensorflow.git)
A real-time hand-tracking and gesture recognition system.
- **Stack:** Python, TensorFlow
- Implements full hand landmark detection and skeletal tracking for gesture-based interaction
- Foundation for touchless UI / HCI applications

#### 👁️ [Eye Cursor with OpenCV](https://github.com/TanishqVarpe/eye_Cursor.git)
An eye-tracking system that translates gaze direction into cursor control.
- **Stack:** Python, OpenCV
- Real-time eye detection and tracking pipeline for accessibility-focused, hands-free computer control
